## Definition

An object in a category is **weakly initial** if there is a morphism from it to every other object, and [[initial]] if this morphism is unique.

This is an instance of a [[weak colimit]].  It is also an instance of a [[weakly initial set]] that happens to be a [[singleton]] set.
