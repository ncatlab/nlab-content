## Idea

A _homotopy sphere_ is a [[topological space]] which need not be [[homeomorphism|homeomorphic]] to an [[n-sphere]], but which has the same [[homotopy type]] as an $n$-sphere. 

## Properties

* Every [[smooth manifold]] homotopy-[[4-sphere]] is [[homeomorphism|homeomorphic]] to the actual [[4-sphere]].

  (due to [[Michael Freedman]], see [Siebenmann](#Siebenmann))
