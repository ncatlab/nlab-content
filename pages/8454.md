## Idea

**LF** ([Pfenning 91](#Pfenning91), [Harper-Honsell-Plotkin 93](#HarperHonsellPlotkin93)) is a certain [[logical framework]] based on [[dependent type theory|dependent]] [[intuitionistic type theory]]. Variants include the _Edinburgh logical framework_, abbreviated "Elf", and _linear LF_, abbreviated LLF ([Pfenning 96](Pfenning96)), the latter capturing also [[dependent linear type theory]].
