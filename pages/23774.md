## Definition 

A **real quadratic function** is a [[quadratic function]] in the [[real numbers]]. 

## Properties

Given a real quadratic function $f(x) \coloneqq a x^2 + b x + c$, if $\vert a \vert \gt 0$, the [[extremum]] occurs at 

$$x = -\frac{b}{2a}$$

and the extremum is a [[minimum]] if $a \gt 0$ and a [[maximum]] if $a \lt 0$. The real quadratic function is a [[linear function]] if $a = 0$.  

## Newton's method

Given a real quadratic function $f:\mathbb{R} \to \mathbb{R}$, $f(x) \coloneqq a x^2 + b x + c$ for real numbers $a:\mathbb{R}$, $b:\mathbb{R}$, $c:\mathbb{R}$, where $\vert a \vert \gt 0$, suppose we want to find an approximate zero of $f$.
