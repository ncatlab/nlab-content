# Idea #



A _[[Michael Batanin|Batanin]] $\omega$-category_ is a [[weak ω-category]] defined as an [[algebra over an operad|algebra]] over the [[globular operad]]. So this is an [[algebraic definition of higher category]].

The definition is similar to that of [[Trimble n-category]].
