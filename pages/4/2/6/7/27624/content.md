## Idea

The concept of a [[terminal object]] in [[simplicial type theory]]. Note that in the context of simplicial type theory, terminal objects make sense for any type, not just the [[Segal types]]. 

## Definition

Let $A$ be a type in [[simplicial type theory]]. An element $y:A$ is a **terminal object** if for all elements $x:A$, the [[hom-type]] $\mathrm{hom}_A(x, y)$ is a [[contractible type]]. 

If $A$ is a [[Segal type]] then this notion coincides with the usual notion of [[terminal object in an (infinity,1)-category]].
