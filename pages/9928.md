## Definition

+-- {: .num_defn}
###### Definition

An even-[[graded ring]] $R$ is a **graded field** if every non-zero homogenous element is multiplicatively invertible.

=--

For instance ([Lurie, lecture 24, def. 1](#LurieLecture)).
