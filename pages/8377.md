## Idea

An [[extensional type theory]] [[proof management system]].

The dialect of [[intuitionistic type theory]] used is also referred to as _[[computational type theory]]_.

The PRL part of the name evidently stands for "Proof Refinement Logic".

## Descendants

* [MetaPRL](http://metaprl.org/): A logical framework implementation that includes (an old version of) Nuprl's logic as a theory, and focuses on an efficient, modular tactic engine for proof automation.

* [RedPRL](http://redprl.org/): An experimental proof assistant for a [[cubical type theory|cubical]] variant of computational type theory.

* [[CompLF|Prototype CompLF]]: A tentative design for a modern, minimalistic variant of Nuprl's logic, with an emphasis on internally reflecting the semantic judgments.
