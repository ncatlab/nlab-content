A set of algebraic operations on a fixed set $S$ is a __clone__
on $S$ if it contains all (component) projections $S^{n}\to S$ and is closed under composition ("superposition").

* Ágnes Szendrei, _Clones in universal algebra_, Séminaire de mathématiques supérieures 99, Les presses de l’université de Montreal, 1986. — 166 p.

A common generalization of a clone and of an operad is proposed, using a new notion of a verbal category, in

* S. Tronin, _Abstract clones and operads_, Siberian Mathematical Journal 43, No.4, 746–755, 2002 [link](http://rd.springer.com/content/pdf/10.1023%2FA%3A1016340806503.pdf)
