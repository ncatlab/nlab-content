
__Sage__ (nowdays SageMath) is a particular free licence _mathematics_ software package. It is interoperable with SINGULAR which comes in a standard package. It uses a version of _python_ interface. It is possible to have a local installation, or use it via an account on the cloud installation (the latter is called the SageMathCloud).
