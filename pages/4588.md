### "twin", a subtype of "pair" ###

A twin is a pair in which both components are the same.

<pre>
t = eval(uneval(p)) // clone "pair"
t.Type['!Atom'] = "twin < pair < structure";
t.part[2] = t.part[1];
t.swapped = t;
</pre>
