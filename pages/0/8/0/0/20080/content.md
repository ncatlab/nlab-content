## Idea

In [[string theory]], _type 0_ refers to [[perturbative string theories]] where the [[strings]] themselves are [[superstrings]] (hence have [[supersymmetry]] on their [[worldsheet]]) but their [[target space|target]] [[spacetime]] [[effective field theory]] is not [[supersymmetry|supersymmetric]], in fact does not even contain any [[fermion fields]].

Technically, this comes from one particular choice of [[GSO projection]] of the [[superstring]] which breaks all target space supersymmetry.
Concretely, the possible [[GSO projection]] for the critical [[NSR superstring]] yield, besides the supersymemtric [[type IIA string theory|type IIA]] and [[type IIB string theory]], two type 0 theories, now called *type 0A* and *type 0B*.
On the other hand, for the [[heterotic string]] there are seven type 0 versions.

All but one of these type 0 theories have a [[tachyon]] mode in the [[closed string]] sector of the standard [[Minkowski spacetime]] [[string theory vacuum|vacuum]] and thus look like [[perturbative string theory|perturbations]] about unstable backgrounds (cf. *[[tachyon condensation]]*). The one exception is the type 0 [[heterotic string]] with [[gauge group]] $O(16) \times O(16)$. ([Dixon & Harvey 1986](#DixonHarvey1986), [Seiberg & Witten 1986](#SeibergWitten1986), [Kawai, Lewellen & Tye 1986](#KawaiLewellenTye86)).

However, it was argued by [Klebanov & Tseytlin 1999a](#KlebanovTseytlin99a) that in nontrivial [[RR field]] [[background field|backgrounds]] the [[tachyon]] mode may disappear, rendering these backgrounds stable, and indeed [[bulk]] [[AdS/CFT duality|duals]] to certain non-supersymmetric [[CFTs]] ([Klebanov & Tseytlin 1999b](#KlebanovTseytlin1999b)).
