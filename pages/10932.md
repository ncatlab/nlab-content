## Idea

An [[lattice]] is _complemented_ if every element has a [[complement]]. It is _orthocomplemented_ if it is equipped with an [[involution]] that sends each element to a complement.

## Examples

* [[Boolean algebra]]

* Birkhoff-vonNeumann [[quantum logic]]
