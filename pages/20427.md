## Idea

A [[duality in string theory]] relating [[heterotic string theory]] to [[type II string theory]] via [[KK-compactification]] on [[K3-surfaces]].

The [[non-perturbative effect|non-perturbative]] enhancement of this is [[duality between heterotic string theory and M/F-theory]].
