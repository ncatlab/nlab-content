## Idea

**$\mathbb{A}^1$-homotopy type theory** or **motivic homotopy type theory** is a hypothetical [[modal type theory|modal]] [[homotopy type theory]] which provides a [[synthetic mathematics|synthetic]] [[foundation]] for [[A1-homotopy theory|$\mathbb{A}^1$-homotopy theory]], where the affine line is the standard [[affine line]] in the [[Nisnevich site]] of [[smooth schemes]] of [[finite type]] over a [[Noetherian scheme]]. 

[[Mitchell Riley]]'s [[bunched logic|bunched]] [[linear homotopy type theory]] &lbrack;[Riley (2022)](dependent+linear+type+theory#Riley22)&rbrack; is a [[synthetic mathematics|synthetic]] foundations for [[parameterized stable homotopy theory|parameterized]] [[stable homotopy theory]]. This may provide a basis for a motivic homotopy type theory. 

## Results

* There is a [[shape modality]] in $\mathbb{A}^1$-homotopy type theory given by the $\mathbb{A}^1$-[[localization of a type|localization]] [[higher inductive type]]. 

* There is no [[cohesion|cohesive]] [[flat modality]] in $\mathbb{A}^1$-homotopy type theory as otherwise that will imply the false statement that the category of $\mathbb{A}^1$-local objects is a [[topos]]. Thus $\mathbb{A}^1$-homotopy type theory is not a [[cohesive homotopy type theory]].
